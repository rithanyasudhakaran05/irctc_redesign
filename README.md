
# IRCTC UI Redesign

##  Overview
This project is a redesign of the IRCTC website to improve usability, visual hierarchy, and overall user experience. The new design focuses on simplicity and ease of booking.

##  Objective
- Simplify the interface  
- Improve navigation  
- Make train search quick and easy  

##  Problems
- Cluttered layout  
- Poor spacing  
- Confusing navigation  

##  Solution
- Clean and minimal UI  
- Centered search section  
- Card-based train results  
- Responsive design for mobile  

##  Screens
- Desktop Design  
- Mobile Design  

##  Tool Used
- Figma
